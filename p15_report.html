Family verdict: the through-line of every result in this report is scale discipline. Ties-to-parents preserves it passively (v1's marathon strength); ties-to-offspring spends it on drift (P15T/P15n's collapse); P16 re-injects it at eviction; P15e spends it FIRST, while it exists, by riding fresh chords at population scale before contracting — repairing f11, the composition function that punished every earlier directed variant (P17: 2900.0, the only sub-PSO result ever recorded there). P17 = P14's oracle-only pair automaton + directed extension + extend-first ordering + farthest re-pairing: affine-hull confined, zero added hyperparameters, best-in-field from G=100 onward on this suite. Remaining open habitats: PSO keeps the G≤50 sprint; structural-tie landscapes (quantized / LLM fitness) are P15T-territory, untested here by construction.
